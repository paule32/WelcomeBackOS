# WelcomeBackOS – ASCII Architecture Diagrams

This package contains diagrams that illustrate the full bootloader and kernel architecture.

## Included Diagrams

- bootflow.txt – Boot chain from BIOS → Stage1 → Stage2 → Kernel
- memorylayout.txt – Memory organization in real & protected mode
- protectedmode.txt – CPU transition from real to protected mode

These diagrams are suitable for documentation, lectures, and OSDev guides.
